# Telegram бот для мониторинга вакансий с hh.ru

## Описание
**Job Hunter Bot** — это Telegram-бот, который помогает пользователям отслеживать новые вакансии на hh.ru. Пользователь настраивает поисковые запросы (например, "TypeScript" в Москве), а бот автоматически присылает уведомления о свежих вакансиях в удобном формате.

## Возможности 
  **Автоматический мониторинг:** Периодическая проверка новых вакансий по подпискам.
*   **Гибкие подписки:** Настройка ключевых слов и региона поиска.
*   **Разовый поиск:** Команда `/find` для мгновенного поиска без подписки.
*   **Интерактивность:** Управление подписками через inline-кнопки.
*   **Красивые уведомления:** Вакансии приходят в виде форматированных карточек с названием, компанией, зарплатой и ссылкой.
*   **Легкое развертывание:** Полная контейнеризация через Docker.

##  Технологический стек

###  Язык и библиотеки
- **TypeScript** — типизация и надежность кода
- **Telegraf** — библиотека для работы с Telegram Bot API
- **Axios** — запросы к API hh.ru
- **node-cron** — запуск периодических задач (поиск вакансий)

### База данных и ORM
- **PostgreSQL** — основное хранилище данных
- **Prisma** — взаимодействие с БД, миграции, типобезопасность

### DevOps и окружение
- **Docker + Docker Compose** — упаковка и развертывание приложения
- **dotenv** — управление переменными окружения




## Использование бота

После запуска найдите бота в Telegram по его username и отправьте команду /start.

### Команды бота

| Команда | Описание |
|---------|---------|
| `/start` | Приветствие и инструкция по использованию |
| `/subscribe` | Создать новую подписку на вакансии |
| `/list` | Показать все активные подписки |
| `/unsubscribe` | Отключить существующую подписку |
| `/find` | Разовый поиск вакансий (без подписки) |
| `/help` | Справка по всем командам |

## Как работает подписка
```text
👤 Пользователь: /subscribe

🤖 Бот:       🔍 Введите ключевые слова для поиска
              (например: TypeScript, Node.js, React)

👤 Пользователь: TypeScript

🤖 Бот:       📍 Выберите регион:
              [ Москва ] [ Санкт-Петербург ]
              [ Новосибирск ] [ Вся Россия ]

👤 Пользователь: [ Москва ]

🤖 Бот:       ✅ Готово! Подписка настроена.
              Буду присылать новые вакансии по запросу "TypeScript" в Москве.
              Уведомления будут приходить автоматически.
```

## Основные сценарии работы

### 1. Создание подписки

1. Пользователь отправляет /subscribe.
2. Бот запрашивает поисковую фразу.
3. Пользователь вводит текст (например, "Node.js").
4. Бот показывает клавиатуру с выбором региона.
5. Пользователь выбирает регион.
6. Бот сохраняет данные в БД и подтверждает подписку.

### 2. Цикл мониторинга (Каждые N минут)

1. Планировщик запускает задачу.
2. Бот загружает из БД все активные подписки.
3. Для каждой подписки отправляется запрос к API hh.ru.
4. Полученные вакансии фильтруются (отсеиваются уже отправленные).
5. Новые вакансии отправляются пользователю.
6. Запись об отправке сохраняется в таблицу notifications.

### 3. Отписка

1. Пользователь отправляет /unsubscribe.
2. Бот показывает список активных подписок (inline-кнопки).
3. Пользователь выбирает подписку для удаления.
4. Бот деактивирует запись в БД.

## Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone <ссылка-на-репозиторий>
   cd TG_Bot_hh_ru
   ```
2. **Настройте переменные окружения (`.env`):**
   Создайте файл `.env` на основе примера:
   ```env
   BOT_TOKEN=ваш_токен
   HH_API_URL=postgresql://user:password@localhost:5432/mydb
   HH_TOKEN=https://api.hh.ru
   ```
3. **Запустите через Docker:**
   ```bash
   docker-compose up --build
   ```
